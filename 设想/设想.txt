符号 模因 现实

符号 {体符， 触符， 范符}
模因 {体模因， 触模因}


规则：
1. 所有的符号都存在一个模因代表自己；
2. 人可以通过符号察觉模因，从而认知符号代表的存在（区别于实在）；
3. 符号的 读取义 不能是自模因，即要读取自模因需要新符号中介；
4. 默认的 读取义 是指符号的义模因；

// 符号体系构建

体符：
自模因 -> 体模因
义模因 -> 符号（自模因）


触符：
自模因 -> 域触模因
义模因 -> 变触模因
主符 -> 符号
次符 -> 符号


范符：
自模因 -> 范模因
义模因 -> 体模因
本符 -> 符号
变符 -> 符号
